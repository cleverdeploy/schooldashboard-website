# schooldashboard-website

Static marketing site for [School Dashboard](https://app.schooldashboard.uk).
Served at https://www.schooldashboard.uk.

## Stack

- Pure HTML + CSS, no build step
- Inter via Google Fonts CDN
- Nginx in a Docker container
- Deployed via [Dokploy](https://dokploy.cleverdeploy.com) on the Hetzner VM
  `dokploy-prod` (178.104.246.72)

## Local preview

```bash
python3 -m http.server 8200
# open http://localhost:8200/
```

## Updating screenshots

Screenshots are generated by `take_screenshots.py` (Playwright). It logs into the
local Rails app at `127.0.0.1:3000`, replaces the school header with a generic
navy banner saying "Oakland High", anonymises all PII (names, emails, VTGs,
phones), then captures each module's page.

```bash
# Reset password on the KES admin user once, then:
python3 take_screenshots.py
cp screenshots/*.png assets/
```

## Deploy

Pushes to `main` auto-deploy via Dokploy.
